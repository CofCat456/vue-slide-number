## Install

> ⚗️ **Experimental**

```
npm i vue-slide-number
```

## Usage

Add `<SlideNumber>` component from `animations-number`.

```html
<script setup>
import { SlideNumber } from 'vue-slide-number'
</script>

<template>
  <SlideNumber :number="1000" /> <!-- here -->
</template>
```

Checkout the [Playground](./playground/) for more examples.
